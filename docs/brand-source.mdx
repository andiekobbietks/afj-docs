## Real logo imagery

Live images, not recreations — the actual master brand mark this entire library derives its tokens from. Drop your own copies into static/img/brand/ .

## Typeface families

No original font files are on record for the master logo, so the closest matching typefaces are documented here for each of the three typographic elements in the lockup.

If an exact match is ever required, hand-tracing the letterforms from the reference PNGs above is the safer route rather than relying on any of these matched fonts, none is a guaranteed pixel-identical substitute for the original.
