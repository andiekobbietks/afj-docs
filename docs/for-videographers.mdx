# For Videographers

What's actually relevant here for motion and video work specifically, not a generic style guide repurposed.

## The existing logo assembly video

The brand audit work already has a real motion asset: the Dynamic Logo Assembly Walkthrough, a looping animation showing the layer order the mark is built from (base letters, rear continent outline, front weave crossing, Cardiff script, tagline). It's delivered in three formats for browser compatibility — WebM (VP9), OGV (Theora), and MP4 (H.264) — a reasonable real-world reference point for web video delivery generally: a modern efficient codec plus a universal MP4 fallback.

## Motion guidance — see "Where Lottie fits"

The one hard rule already established for motion elsewhere on this site applies here too: **loop ambient or idle animation only** — never loop a success or confirmation moment. It should play once and land, or it stops reading as confirmation and starts reading as decoration. If cutting a sting or bumper that ends on the logo lockup, that's a "confirmation" moment in this sense — let it settle rather than looping.

## Framing, by which design system the footage is supporting

The Scrollytelling system was built deliberately cinematic — near-black canvas, wide negative space, slow reveals — so footage cut for that context (the pre-launch teaser, brand storytelling) reads correctly in a widescreen, letterboxed, patient pace.

The Wine/Gold system is the transactional, calmer register — footage supporting booking or product flows should be cut tighter and more direct, closer to a product demo than a mood piece. See Compare & Contrast on the [Component Library](./component-library) page for the fuller reasoning behind why the two registers exist at all.

**Colour note:** the same RGB-vs-delivery-format caveat as print applies here in spirit — screen-sampled hex values may need grading adjustment depending on the final delivery platform's colour handling. Confirm on the actual delivery screen or platform rather than trusting the hex code alone.
