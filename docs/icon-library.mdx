# Icon Library

Icons are [Lucide](https://lucide.dev) (ISC licensed — free for commercial use, no attribution required) — the working icon-set decision made directly in the canonical component library doc, chosen for its consistent 24×24 grid and active maintenance rather than mixing icon sets ad hoc. Every icon is inline SVG with `stroke="currentColor"`, so it tints automatically per theme with no separate icon-colour token needed.

A live, themeable preview (Wine/Gold, Scrollytelling, Day mode, Scrolly Day) lives at [`/icon-library.html`](/icon-library.html), embedded inline at the bottom of this page.

**Source:** `icon-library.html`, generated from the real `lucide-static` npm package (not hand-drawn approximations).
