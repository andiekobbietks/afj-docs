# Core UI

Buttons and the product card — the base primitives everything else assembles from.

import DemoButtons from '@site/src/components/DemoButtons';
import DemoProductCard from '@site/src/components/DemoProductCard';

## 1. Button component

Radius switches from 8-10px in Wine/Gold and Day mode to a full pill in Scrollytelling via `--btn-radius`. Real Bootstrap 5 classes — no bespoke `.demo-btn` styling, this is the same markup that would work on afjcardiff.com directly (no React needed there):

<DemoButtons />

```html
<button type="button" class="btn btn-primary">Book a class</button>
<button type="button" class="btn btn-outline-primary">View basket</button>
```

```css
/* Themes Bootstrap's own component variables, not a parallel system */
[data-afj-theme] .btn-primary {
  background: var(--gradient);
  border-color: transparent;
  border-radius: var(--btn-radius);
}
```

## 2. Card component, shared-border frame
