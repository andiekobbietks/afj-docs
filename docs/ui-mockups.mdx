## On the noir/near-black tokens

There's no single token named "noir" — it's a descriptive phrase from the working session, not a color. The four real near-black values in the system each do a different job:

- `--bg-base` (`#09080C`) — the page canvas, Wine/Gold and Bootstrap dark theme
- `--color-bg-deep` (`#0d0c11`) — a third, darker step for recessed elements
- `--bg-offset` (`#141116`) — one step up from the canvas, for cards and panels, so they read as sitting *on* the surface rather than flattened into it
- Scrollytelling canvas (`#050507`) — its own deliberately darker value, serving that theme's cinematic register specifically

All four sit within about 2.5 points of pure black's maximum 21:1 contrast against white text, so none of them is a contrast compromise — the softer values buy comfort (avoiding the halation a true `#000` can produce next to white), not compliance. Each also carries a faint warm tint toward violet/plum rather than true neutral grey, the same wine hue bleeding into the canvas.
