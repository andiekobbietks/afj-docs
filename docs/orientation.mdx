## About this site

AFJ Cardiff is Wales' first dedicated African dance organisation, founded by Plamedi Santima-Akiso. This site is the internal component library and decision record for AFJ Cardiff's digital properties, the merch shop, class booking, course subscriptions, and the pre-launch scrollytelling teaser.

Three groups use this site for different reasons: engineers copy component markup and read the ADR log before making a change; designers and marketing check that a colour, radius, or piece of copy is on-brand before it ships; and anyone new to the project uses Start Here and the Glossary to get oriented without needing to ask. Nothing here is customer-facing, that's why the repo should stay private for now.
