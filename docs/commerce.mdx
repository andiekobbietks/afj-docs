# Commerce

Cart drawer and class booking card. For the full working shop and booking flows, see [Customer Journeys](./customer-journeys) — this page shows just these two pieces in isolation.

import CartDrawerDemo from '@site/src/components/CartDrawerDemo';

## Cart drawer

Real Bootstrap Offcanvas markup, not a custom slide-in panel:

<CartDrawerDemo />

```html
<div class="offcanvas offcanvas-end" tabindex="-1" id="cartDrawer">
  <div class="offcanvas-header">
    <h5 class="offcanvas-title">Your basket</h5>
    <button type="button" class="btn-close" data-bs-dismiss="offcanvas"></button>
  </div>
  <div class="offcanvas-body">...</div>
</div>
```

```js
// Vanilla JS: Bootstrap's own API, no custom transform/transition CSS needed
new bootstrap.Offcanvas('#cartDrawer').show();
```
