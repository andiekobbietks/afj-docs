## 4. Carousel component

**What Bootstrap doesn't provide here, and how it's built anyway:** Bootstrap's own `.carousel` is a single-slide auto-rotating slideshow — a genuinely different thing from this row, which shows multiple cards at once and scrolls by drag. Using `.carousel` here would be the wrong component for the job, not a shortcut. Real Bootstrap still does most of the work (`overflow-x-auto`, `d-flex`, `gap-2` are all real utilities); only the drag-to-scroll mouse handling is vanilla JS, since no framework — Bootstrap included — ships that as a utility:

<CarouselDemo />

```html
<div class="d-flex overflow-x-auto gap-2">...</div>
```

## 6. Tonal scale explainer

How the seven derived shades of the base accent actually get used across UI states — reads `--accent-2` and derives the seven stops from it via `color-mix()`, so it's the wine scale in Wine/Gold, the orange→red scale in Scrollytelling, and the same wine scale again (just on beige) in Day mode:

<TonalScaleDemo />

```css
/* color-mix() is standard CSS, not Bootstrap-specific — no utility class
   exists for "N% of a custom property" in any framework */
background: color-mix(in srgb, var(--accent-2) 65%, var(--bg-base));
```
