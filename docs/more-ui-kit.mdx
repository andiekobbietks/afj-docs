## 3. Gallery grid

4 columns desktop, hover scales a tile to 1.03, collapses to 2 columns under 500px. The simplest component in the whole kit, on purpose.

**Preview**

```css
.afj-gallery-grid { display: grid; grid-template-columns: repeat(4, 1fr); }
@media (max-width: 500px) {.afj-gallery-grid { grid-template-columns: 1fr 1fr; } }
```

## 4. Carousel component

overflow-x: auto handles native touch swipe, mouse-drag added via JS cursor feedback, no extra CSS needed for that part.

**Preview**

```css
.afj-testimonial-row { display: flex; overflow-x: auto; cursor: grab; }
```

## 5. Interactive input

Bordered, transparent fill, stays visually quiet as a secondary action next to primary CTAs. Scales to 40px height on mobile for tap ease.

**Preview**

```css
.afj-newsletter { background: transparent; border: 1px solid var(--border); }
@media (max-width: 500px) {.afj-newsletter { height: 40px; } }
```

## 6. Tonal scale explainer

How the seven derived shades of the base accent actually get used across UI states. This table is token-driven, it reads --accent-2 and derives the seven stops from it, so it's the wine scale in Wine/Gold, the orange→red scale in Scrollytelling, and the same wine scale again (just on beige) in Day mode. One rule, three renders.
