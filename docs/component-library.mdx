# Component Library

This page documents the canonical AFJ Cardiff components. Each entry lists the live preview source, the token dependencies it pulls from Design System, and a copyable snippet.

A live, themeable preview of every component below (toggle between Wine/Gold, Scrollytelling, Day mode, and Scrolly Day) lives at [`/component-library.html`](/component-library.html), embedded inline further down this page.

---

## Buttons

| Property | Value | Token |
| --- | --- | --- |
| Radius | `8–10px` (Wine/Gold, Day) / `50px` pill (Scrollytelling) | `--btn-radius` |
| Primary fill | Brand gradient | `--gradient` |
| Outline | 1px solid current text colour | `--text` |

```css
.demo-btn.primary { background: var(--gradient); border-radius: var(--btn-radius); }
.demo-btn.outline { background: transparent; border: 1px solid var(--text); }
```

**Source:** `component-library.html`

## Card — shared-border frame

The card is the base unit for products, classes, and gallery tiles. Badge colour always pulls `--accent-2` across all themes.

```css
.demo-card { background: var(--bg-offset); border: 1px solid var(--border); border-radius: var(--radius); }
.demo-badge { background: var(--accent-2); color: #fff; }
```

**Source:** `component-library.html`

## Gallery grid

4 columns desktop, collapses to 2 under 500px. Hover scales a tile to `1.03`.

```css
.afj-gallery-grid { display: grid; grid-template-columns: repeat(4, 1fr); }
@media (max-width: 500px) { .afj-gallery-grid { grid-template-columns: 1fr 1fr; } }
```

**Source:** `component-library.html`

## Tonal scale

Seven derived shades of the base accent (`--accent-2`), used consistently for hover/pressed/disabled states across every theme — never a second, unrelated hue.

| Stop | Relation | Usage |
| --- | --- | --- |
| Darkest &minus;40% | `color-mix(in srgb, var(--accent-2) 60%, black)` | Pressed button backgrounds |
| Darker &minus;30% | `70%, black` | Card borders and dividers on dark surfaces |
| Dark &minus;20% | `80%, black` | Hover state on primary buttons and links |
| Base | `var(--gradient)` | Primary buttons, links, focus indicators |
| Light +15% | `85%, white` | Secondary badges, tag fills |
| Lighter +25% | `75%, white` | Icon fills on dark backgrounds |
| Lightest +40% | `60%, white` | Text on dark fills |
